DBMS Lab :-

This repository contains DBMS lab experiments and SQL queries performed as part of coursework.

# Overview

The repository includes practical implementations of database concepts using SQL. It covers basic to intermediate operations performed during lab sessions.

# Technologies Used

* SQL
* XAMPP (MySQL / phpMyAdmin)

# Topics Covered

* SQL Queries
* Data Definition Language (DDL)
* Data Manipulation Language (DML)
* Joins
* Constraints
* Basic database operations

# How to Use

* Open XAMPP and start Apache & MySQL
* Use phpMyAdmin or MySQL to run SQL queries
* Browse the files in this repository and execute them
